前端访问说明

要访问完整系统，请按以下步骤启动前端服务：

  1. 打开新的命令行窗口
  2. 进入前端目录：cd frontend
  3. 安装依赖：npm install
  4. 启动开发服务器：npm run dev
  5. 访问：http://localhost:3002

API接口说明

后端API基础路径：http://localhost:8081/springmvc030/api

例如登录接口：http://localhost:8081/springmvc030/api/doLogin
